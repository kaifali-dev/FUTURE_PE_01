# FUTURE_PE_01 - AI Website Copy Generator for Local Businesses

## Project Overview

This project was completed as part of the Prompt Engineering Internship at Future Interns.

The objective of this project is to design a structured prompt system that generates high-quality website copy for local businesses. The generated content includes homepage copy, service descriptions, customer-focused messaging, and conversion-oriented call-to-action sections.

The prompt framework was designed to create business-specific website content that can be adapted for different industries and clients.

---

## Business Chosen

**Business Name:** RACE

## Prompt Logic

The prompt was structured to include:

* Business Information
* Target Audience
* Unique Selling Proposition
* Website Goals
* Required Website Sections
* Writing Style Requirements

This approach helps generate business-specific and conversion-focused website content rather than generic AI-generated text.
